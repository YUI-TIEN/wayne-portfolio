# Wayne Portfolio

Personal portfolio site for Yuwei "Wayne" Tien, focused on AI product landing,
agent workflow operations, runtime diagnostics, and demo/POC delivery systems.

## Stack

- React
- TypeScript
- Vite
- GitHub Pages
- GitHub Actions

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push to `main`. The GitHub Actions workflow builds the Vite app and deploys the
`dist` artifact to GitHub Pages.
